
< 전체 게임 화면 진행 >
1. 타이틀 화면(게임시작버튼 -> 캐릭터선택 창 -> 맵 선택 창)
2. 게임 화면(플레이어, 몬스터, 아이템, 게임오버 창, 타이머, ...)
3. 종료 화면(게임 결과, 완료버튼 누르면 다시 타이틀 화면으로)


< 화면별 클래스 정리 >
[ 타이틀 화면 ]
"아무키나 누르세요" -> 게임시작 버튼 클릭 -> 캐릭터 선택 -> 맵 선택
	
	# 배경화면
	  일단 배경화면만 띄우고 버튼 누르면 게임 화면으로 전환되는 기능을 먼저 만들예정
	  @ 사용자 입력 기능
	    - 스페이스바(확인), 방향키(메뉴선택)
		- 버튼 클릭(게임시작 버튼, ...)

[ 게임 화면 ]
	
	# 플레이어
	  @ 상하좌우 이동 기능

	  @ 충돌 기능
		- 적과 충돌
		- 아이템과 충돌
		- 지형과 충돌

	  @ 기타 입력(스페이스바: 확인, esc: 일시정지및 스탯창)

	  @ 플레이어 스탯 (우선 * 부터 구현할 예정)
	    - 체력*
		- 회복량
		- 방어력
		- 이동속도*
		- 괴력
		- 공격범위
		- 투사체 속도
		- 지속시간
		- 투사체 수 (max : 2)
		- 쿨타임
		- 행운
		- 성장
		- 탐욕
		- 저주
		- 자석
		- 부활
		- 무기***
			무기 종류(채찍, 룬트레이서, 매직완드, 나이프, 도끼, 성경책, 파이어완드, 마늘, 시계바늘, 뼈다귀, 십자가)
	  @ 플레이어 패시브 (무기를 업그레이드 해주는 능력)
	    - UI
		  화면 왼쪽 위 배치
		- 괴력, 방여력, 최대 체력, 회복량, 쿨타임, 공격범위, 투사체 속도, 지속시간, 투사체 수, 이동속도,
		  자석, 행운, 성장, 탐욕, 저주, 부활

	# 몬스터
	  @ 일반 몬스터
	    
	  @ 보스 몬스터
	    - 죽으면 보물상자 떨굼

	# 아이템
	  @ 코인

	  @ 화염 방사기

	  @ etc

	# 타이머
	  - 00 : 00 부터 시작하는 타이머 기능
	  - 일시정지하면 타이머 멈춤

[ 종료 화면 ]
