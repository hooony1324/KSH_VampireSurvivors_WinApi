## 조작법

WASD : 이동<br/>
SpaceBar : 선택<br/>
ESC : 일시정지<br/>
1, 2, 3, 4 : 레벨업 스킬 선택<br/>
L : 레벨업 치트<br/>
C : 충돌체 활성화<br/>
G : 무적 모드<br/>
R : 리퍼 소환<br/>
M : 충돌맵 활성화<br/>
F : 시간 빠르게 감기 모드<br/>


## 플레이 방법
타이틀 화면에서 "아무 키나 누르세요" 상태에서 SpaceBar로 화면을 넘어가시면서 플레이 하시면 됩니다.
